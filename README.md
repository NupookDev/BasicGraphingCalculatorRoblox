## How to write equation:
This calculator uses Reverse Polish Notation.
### Example:
"y = sin(3 * x)" &rarr; "y = 3 x * sin"<br/>
"y = 3 + 4 + x" &rarr; "y = 3 4 + x +"<br/>
"y = 1/x" &rarr; "y = 1 x /"
